# DISC-тест кандидата — RCode / Mirro

Тест DISC для кандидатов на позицию преподавателя/наставника: 28 утверждений,
автоматический подсчёт баллов по четырём стилям (D/I/S/C) и мгновенный вывод
результата. Вся логика — на клиенте, никакого сервера для самого теста не нужно.

Сайт: https://fbismark1717-work.github.io/dsci-disc-test/

## Сохранение результатов в Google Таблицу

Результат каждого прохождения можно автоматически дописывать строкой в Google
Таблицу через Apps Script Web App — без токенов в коде страницы.

1. Создайте новую Google Таблицу (sheets.new), например «DISC — результаты кандидатов».
2. **Extensions → Apps Script**, удалите содержимое по умолчанию и вставьте код
   из [`apps-script/Code.gs`](apps-script/Code.gs) этого репозитория.
3. **Deploy → New deployment → Тип: Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
   - Deploy, разрешить запрошенные права
4. Скопируйте URL веб-приложения (заканчивается на `/exec`).
5. В `index.html` найдите строку `var RESULTS_ENDPOINT = "";` и вставьте туда
   этот URL.
6. Закоммитьте и запушьте — заголовки колонок таблица создаст сама при первом
   прохождении теста.

Пока `RESULTS_ENDPOINT` пустой, результаты нигде не сохраняются — только
показываются кандидату на экране.

## Публикация через GitHub Pages

1. Settings → Pages → Source: **Deploy from a branch**
2. Branch: `master`, папка `/ (root)`
3. Сохранить — через 1–2 минуты сайт появится по адресу
   `https://<username>.github.io/<repo-name>/`

## Локальный запуск

Открыть `index.html` в браузере — файл полностью самодостаточный (кроме,
опционально, отправки результата в Google Таблицу).
