# Тест типологии личности DISC — D.S.C.I.

Самостоятельный тест DISC для сотрудников: 28 утверждений, автоматический подсчёт баллов
по четырём стилям (D/I/S/C) и мгновенный вывод результата — без сервера и без отправки
данных куда-либо, всё считается в браузере на JavaScript.

## Публикация через GitHub Pages

1. Settings → Pages → Source: **Deploy from a branch**
2. Branch: `main`, папка `/ (root)`
3. Сохранить — через 1–2 минуты сайт появится по адресу
   `https://<username>.github.io/<repo-name>/`

## Локальный запуск

Открыть `index.html` в браузере — файл полностью самодостаточный.
